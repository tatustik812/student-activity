# 📊 Анализ активности студентов и A/B-теста

## Результаты
- Пик активности: вечер (17–23 ч) и пятница.
- Просмотры новостной ленты коррелируют с числом коммитов (коэффициент ~0.6).
- A/B-тест: после внедрения новостной ленты тестовая группа стала сдавать работы раньше дедлайна.
- Анимация процесса сдачи `project1` показывает накопление попыток каждого студента.

## Технологии
Python, pandas, numpy, matplotlib, seaborn, plotly, sqlite3, Jupyter.

## Структура
- `00_line_chart.ipynb` – ежедневная активность
- `01_line_chart_styles.ipynb` – просмотры vs коммиты
- `02_bar_chart.ipynb` – активность по частям суток
- `03_bar_charts.ipynb` – будни vs выходные
- `04_histogram.ipynb` – распределение коммитов по часам
- `05_boxplot.ipynb` – A/B-тест (лаг до/после)
- `06_scatter_matrix.ipynb` – корреляции
- `07_heatmap.ipynb` – тепловые карты (пользователи × дни/часы)
- `08_seaborn.ipynb` – динамика попыток project1
- `09_plotly.ipynb` – интерактивная анимация
